# Acceptance test contracts

One file per goal: `G-xxx.md`. Gives agents **testable** expected behavior — not vibes.

## When to create

- When promoting a goal from `draft` → `ready`
- Required for user-facing or API goals; optional for pure infra bootstrap

## Template

Copy [_template.md](_template.md) → `G-xxx.md` and fill every section.
